# 📘 Kesimpulan Materi (Tugas 4, 5, 6)

Materi yang dibahas mencakup struktur data, fungsi, OOP, serta pengolahan data menggunakan library Python.

Struktur data seperti list, tuple, set, dan dictionary memiliki karakteristik berbeda dalam penyimpanan dan manipulasi data. List bersifat fleksibel, tuple bersifat immutable, dan set hanya menyimpan data unik tanpa duplikat. Selain itu, teknik slicing (`data[start:stop:step]`) memungkinkan pengambilan data secara efisien.

Function digunakan untuk membuat kode yang modular dan reusable, sedangkan konsep Object-Oriented Programming (OOP) memungkinkan pengelolaan data dan perilaku dalam bentuk objek yang terstruktur.

Penggunaan NumPy dan pandas memperluas kemampuan Python dalam pengolahan data, mulai dari perhitungan statistik hingga pengelolaan data dalam bentuk tabel. Integrasi dengan File I/O memungkinkan penyimpanan hasil pengolahan data, sehingga dapat digunakan kembali.

Secara keseluruhan, materi ini membentuk dasar penting dalam pemrograman Python, khususnya untuk pengolahan data dan pengembangan sistem berbasis data.
